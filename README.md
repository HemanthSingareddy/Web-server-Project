## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for development)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/HemanthSingareddy/Web-server-Project.git
   cd Web-server-Project
   ```

2. **Navigate to the project directory**
   ```bash
   cd no-framework
   ```

3. **Open the application**
   - Simply open `index.html` in your web browser, or
   - Use a local web server for better performance:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js http-server
     npx http-server
     ```

4. **Access the application**
   - If using a web server, navigate to `http://localhost:8000` in your browser
   - If opening directly, double-click `index.html`
